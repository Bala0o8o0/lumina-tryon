# AI E-Commerce Virtual Try-On

This is a Next.js project for an AI-powered e-commerce virtual try-on application. The goal of this project is to create a platform where users can virtually try on clothing and accessories using their webcam or by uploading a photo.

For a comprehensive guide to the project, including its structure, technologies, and development conventions, please see the [GEMINI.md](GEMINI.md) file.

## Features

*   **Virtual Try-On:** Users can virtually try on clothing and accessories using their webcam or by uploading a photo.
*   **AI-Powered Recommendations:** The application will use AI to provide personalized recommendations based on the user's style and preferences.
*   **E-Commerce Integration:** The application will be integrated with an e-commerce platform to allow users to purchase the items they try on.
*   **Social Sharing:** Users can share their virtual try-on photos with friends on social media.

## Tech Stack

*   **Framework:** [Next.js](https://nextjs.org/)
*   **Language:** [TypeScript](https://www.typescriptlang.org/)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
*   **UI:** [React](https://reactjs.org/)

## Getting Started

To get the project up and running on your local machine, follow these steps:

**1. Prerequisites:**

*   Node.js (v20 or later recommended)
*   npm (or your preferred package manager)

**2. Clone the Repository:**

```bash
git clone <repository-url>
cd <project-directory>
```

**3. Install Dependencies:**

```bash
npm install
```

**4. Run the Development Server:**

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.
